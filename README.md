# Discogs Vinyl Analysis

Finding undervalued vinyl records using machine learning.

## Data

**Source**: Discogs monthly data dump (January 2026)
**Total releases**: 18,797,505

Download the sample data from Dropbox: [link coming soon]

Place the file in the `data/` folder before running the notebooks.

## Setup
1. Add your Discogs API token to `.env`
2. `pip install -r requirements.txt`

## Structure
- `data/` — raw and processed data (not in git)
- `src/` — scripts
- `notebooks/` — analysis
- `outputs/` — charts
